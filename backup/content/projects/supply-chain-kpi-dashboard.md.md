# Supply Chain KPI Dashboard

## Problem

Operations teams often lack a unified view of performance across key metrics such as safety, quality, service, and cost.

## Objective

Build a KPI-driven dashboard that enables:

- Real-time visibility into performance  
- Faster identification of issues  
- Data-driven decision-making  

## Approach

- Designed KPI framework aligned to operational goals  
- Structured data using SQL queries  
- Built dashboard views for performance tracking  
- Integrated trend and variance analysis  

## Key Metrics

- Defect rates  
- Throughput  
- On-time delivery  
- Inventory accuracy  
- Productivity  

## Business Impact

- Improved visibility and accountability  
- Faster countermeasure development  
- Better alignment between teams
