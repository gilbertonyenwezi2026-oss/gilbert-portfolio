# Go + Hugo Portfolio Website

## Objective

Build a fast, scalable personal website using Hugo to showcase technical projects and professional experience.

## Approach

- Used Hugo static site generator  
- Structured content using Markdown  
- Applied Go templating via Hugo themes  
- Deployed via GitHub + Netlify  

## Features

- Portfolio-driven design  
- Fast load times  
- Structured navigation  
- Git-based deployment workflow  

## Business Value

- Professional online presence  
- Clear demonstration of technical capability  
- Scalable platform for future projects  

## Tools

Go, Hugo, Git, GitHub, Netlify
